AI Playerbot by ike3

This is a fork of playerbot with completely different AI based on strategies.

Welcome to the AI Playerbot project! This is a modification of MaNGOS server which brings a number of bots online and available for any player. Also allows you to use your account/guild characters as bots. This includes:

  • Easy bot control using chat commands
  • About 100 bots online spread around the world at any time waiting for your invitation
  • More than 100 chat commands to interact with the bots
  • More than 30 strategies to choose from
  • Improved auction house bot
  • Structured and maintainable source code.

This makes it possible the use cases below:

  • Inviting bots to party and playing dungeon, raid or doing some PvP
  • Summoning your alts as bots to quick item exchange, casting some spells, buffing, leveling, etc.
  • Random PVP with bots of opposite faction or dueling with bots from your faction
  • Trading and auctioning
  • Making World of Warcraft completely a single-player game!

This project is an alternative for Blueboy's playerbot (actually AI Playerbot was forked from it but the codebase is now almost fully rewritten and incompatible with the original version).

Download

You can download this project in either zip or tar formats. You can also clone the project with Git by running:

Support

AI Playerbot is known to be working on the following mangos versions:

  • MaNGOS R2 (main branch)
  • MaNGOS Zero
  • MaNGOS One
  • MaNGOS Two
  • CMaNGOS
There are separate branches on GitHub for each version. New features are being developed on R2 branch until a release is done. Then the release is ported to other versions.
